Detailed Description

几何对象关联的属性场。 @detials 属性场的定义域可以是几何对象的顶点集、边集、面片集或者体元集；其值域可以包含多个属性字段。属性字段规定了其取值类型、无效值等信息。 ShapeProperty 的值域支持布尔值、整数、浮点数，以及字符串类型的属性字段。
