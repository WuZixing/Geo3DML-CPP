Detailed Description

可视化材质参数。 地调局标准（简称局标，或v1版标准）与地理信息协会标准（简称协会标准，或v2版标准）使用不同的材质定义。 两个版本使用完全不同的属性。但从实际效果看，v2版本涵盖了v1版本。
