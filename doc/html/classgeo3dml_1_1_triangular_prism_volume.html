指定构成体元的6个顶点，添加一个三棱柱体元。

Parameters
top1,top2,top3构成顶面三角形的3个顶点的索引号。顶点的顺序应保持该面在右手坐标系下的法向朝向体元外部。
bottom1,bottom2,bottom3构成底面三角形的3个顶点的索引号。顶点的顺序使得top1、top2、bottom1与bottom2对应构成三棱柱的一个侧面；其余顶点依此类推。
Returns
返回新添加的体元的索引号。
