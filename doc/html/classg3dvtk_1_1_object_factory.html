vecJ网格J坐标轴对应于全局坐标系的单位向量，表达网格单元沿J轴的棱的方向与长度。
vecK网格K坐标轴对应于全局坐标系的单位向量，表达网格单元沿K轴的棱的方向与长度。
highI在局部坐标系下，网格的最大I坐标。
highJ在局部坐标系下，网格的最大J坐标。
highK在局部坐标系下，网格的最大K坐标。
lowI在局部坐标系下，网格的最小I坐标。缺省值为0。
lowJ在局部坐标系下，网格的最小J坐标。缺省值为0。
lowK在局部坐标系下，网格的最小K坐标。缺省值为0。
Returns
